An agent‐powered 911 dispatch system

Description

Traditional 911 systems rely on manual assessment, causing slower response times during critical emergencies. So we built an agentic pipeline integrating AWS Connect, Transcribe, Comprehend, and Bedrock to automate emergency call processing. The system performed real-time transcription → PII redaction → sentiment analysis → AI-generated action plans for first responders, all with sub-second latency. Technical Highlights: Seamless AWS service integration for live call streaming, HIPAA-compliant PII scrubbing using Comprehend, Bedrock-powered agentic reasoning to generate contextual response plans, Real-time processing pipeline optimized for time-critical decisions.

Outcome: Reduced simulated response time by 1 minute and increased operational effectiveness 3x. Won 1st place among 150+ participants at AWS Breaking Barriers Hackathon, demonstrating how agentic AI can transform critical infrastructure while maintaining security and compliance standards.

Tech stack

TypeScript React Node.js PostgreSQL AWS Docker
